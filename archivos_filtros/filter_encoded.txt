<span class = 'colorClave'><b>FILTER_SANITIZE_ENCODED</b></span><br><br>
Remueve o encodea los caracteres especiales en la url provista. Retorna la url filtrada.<br><br>
<b>BANDERAS</b><br>
<i class = "colorSubtitulo">FILTER_FLAG_STRIP_LOW</i>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;: Limpia todos los caracteres con un valor <32.<br>
<i class = "colorSubtitulo">FILTER_FLAG_STRIP_HIGH</i>&nbsp;&nbsp;&nbsp;&nbsp;: Limpia todos los caracteres con un valor >127.<br>
<i class = "colorSubtitulo">FILTER_FLAG_STRIP_BACKTICK</i>: Limpia todos los caracteres '`'.<br>
<i class = "colorSubtitulo">FILTER_FLAG_ENCODE_LOW</i>&nbsp;&nbsp;&nbsp;&nbsp;: Encodea todos los caracteres con un valor <32.<br>
<i class = "colorSubtitulo">FILTER_FLAG_ENCODE_HIGH</i>&nbsp;&nbsp;&nbsp;: Encodea todos los caracteres con un valor >127.<br>
