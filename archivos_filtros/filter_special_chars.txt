<span class = 'colorClave'><b>FILTER_SANITIZE_SPECIAL_CHARS</b></span><br><br>
Filtra los caracteres especiales <i>HTML</i>. Es utilizado para escapar los caracteres "<>& y con valor <i>ASCII</i> menor a 32.
Retorna el valor filtrado.<br><br>
<b>BANDERAS</b><br>
<i class = "colorSubtitulo">FILTER_FLAG_STRIP_LOW</i>&nbsp;&nbsp;: Limpia todos los caracteres con un valor <32.<br>
<i class = "colorSubtitulo">FILTER_FLAG_STRIP_HIGH</i>&nbsp;: Limpia todos los caracteres con un valor >127.<br>
<i class = "colorSubtitulo">FILTER_FLAG_ENCODE_HIGH</i>: Encodea todos los caracteres con un valor >127.<br>
