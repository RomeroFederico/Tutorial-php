<span class = 'colorClave'><b>FILTER_VALIDATE_REGEXP</b></span><br><br>
Valida el valor contra una expresion regular <i>'regexp'</i>, retornando el valor si cumple, <i>false</i> en caso contrario.<br><br>
<b>OPCIONES</b><br>
<i class = "colorString">regex&nbsp;&nbsp;</i>: Expresion regular a la que se somete el valor ingresado.<br>
<i class = "colorString">default</i>: En caso de no poder validar, retorna este valor.<br><br>
<b>BANDERAS</b><br>
<i class = "colorSubtitulo">FILTER_NULL_ON_FAILURE</i>: Retorna <i>null</i> en caso de no poder validar.<br>
